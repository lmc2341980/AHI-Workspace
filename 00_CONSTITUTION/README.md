AHI does not aim to replace human intelligence.

```
Human Intelligence
        +

Artificial Hybrid Intelligence
```

- collaboration;
- co-learning;
- co-evolution;
- knowledge inheritance;
- long-term value creation.

# 4. Constitutional Principles

## Human Ownership

Human remains the final decision maker.

AI supports:

- analysis;
- generation;
- validation;
- optimization.

---

## Inheritance First

No Artifact is created from zero.

Every evolution must define:

- inherited knowledge;
- preserved principles;
- improvements;
- additions.

---

## Truth Separation

AHI distinguishes:

```
FACT

RESTORE

UNKNOWN

PROPOSAL
```

No assumption becomes official knowledge without validation.
