# 1. Purpose

AHI Instruction Set Architecture (AHI-ISA) định nghĩa tập lệnh chuẩn của toàn bộ hệ sinh thái Artificial Hybrid Intelligence.

AHI-ISA không phải Command Line.

AHI-ISA là Semantic Instruction Set.

Mục tiêu:

- giảm thao tác;
- giảm số ký tự;
- giảm số lần copy;
- giảm số lần mô tả;
- tăng khả năng tái sử dụng;
- tăng khả năng tự động hóa;
- chuẩn hóa cộng tác giữa người và AI.

# 11. Human First

Instruction chỉ giúp giảm thao tác.

- Human Ownership
- Human Consent
- AHI Constitution
