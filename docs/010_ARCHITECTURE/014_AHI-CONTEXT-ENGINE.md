# Purpose (Mục đích)

This document defines the Context Engine of the Artificial Hybrid Intelligence Workspace (AHI-Workspace - Không gian làm việc Trí tuệ Nhân tạo Lai).

(Tài liệu này định nghĩa Bộ máy Ngữ cảnh của Không gian làm việc Trí tuệ Nhân tạo Lai.)

---

# Vision (Tầm nhìn)

AHI does not understand conversations.

(AHI không hiểu các cuộc hội thoại.)

AHI understands contexts.

(AHI hiểu các ngữ cảnh.)

A conversation is only one possible representation of a context.

(Một cuộc hội thoại chỉ là một cách biểu diễn của ngữ cảnh.)

Contexts evolve continuously.

(Ngữ cảnh liên tục tiến hóa.)

---

# Core Responsibilities (Các chức năng cốt lõi)

The Context Engine is responsible for:

(Bộ máy Ngữ cảnh chịu trách nhiệm:)

- Building Contexts (Xây dựng Ngữ cảnh)
- Merging Contexts (Hợp nhất Ngữ cảnh)
- Splitting Contexts (Tách Ngữ cảnh)
- Searching Contexts (Tìm kiếm Ngữ cảnh)
- Ranking Contexts (Xếp hạng Ngữ cảnh)
- Linking Contexts (Liên kết Ngữ cảnh)
- Evolving Contexts (Tiến hóa Ngữ cảnh)
- Archiving Contexts (Lưu trữ Ngữ cảnh)

# Context Structure (Cấu trúc Ngữ cảnh)

Every context shall contain:

(Mỗi ngữ cảnh bao gồm:)

- Context Identifier (Định danh Ngữ cảnh)
- Semantic Identifier (Định danh Ngữ nghĩa)
- Owner (Chủ sở hữu)
- Participants (Thành phần tham gia)
- Related Contexts (Các Ngữ cảnh liên quan)
- Knowledge Objects (Các Đối tượng Tri thức)
- Memory Layer (Tầng Bộ nhớ)
- Permissions (Phân quyền)
- Evolution History (Lịch sử Tiến hóa)

---

# Context Relationships (Quan hệ Ngữ cảnh)

A context may relate to unlimited other contexts.

(Một ngữ cảnh có thể liên kết với vô số ngữ cảnh khác.)

Examples include:

(Ví dụ:)

- Parent Context (Ngữ cảnh Cha)
- Child Context (Ngữ cảnh Con)
- Parallel Context (Ngữ cảnh Song song)
- Reference Context (Ngữ cảnh Tham chiếu)
- Personal Context (Ngữ cảnh Cá nhân)
- Organizational Context (Ngữ cảnh Tổ chức)
- Global Context (Ngữ cảnh Toàn cục)

---

# Context Evolution (Tiến hóa Ngữ cảnh)

The Context Engine continuously improves context quality through:

(Bộ máy Ngữ cảnh liên tục nâng cao chất lượng ngữ cảnh thông qua:)

- Semantic Learning (Học Ngữ nghĩa)
- User Feedback (Phản hồi Người dùng)
- Artificial Intelligence Collaboration (Cộng tác Trí tuệ Nhân tạo)
- Memory Optimization (Tối ưu Bộ nhớ)
- Relationship Discovery (Khám phá Quan hệ)
- Knowledge Consolidation (Hợp nhất Tri thức)

---

# Architectural Principles (Nguyên tắc Kiến trúc)

1. Context is permanent.

(Ngữ cảnh tồn tại lâu dài.)

2. Meaning is more important than chronology.

(Ngữ nghĩa quan trọng hơn trình tự thời gian.)

3. Context may evolve forever.

(Ngữ cảnh có thể tiến hóa vô hạn.)

4. Multiple Artificial Intelligence systems may collaborate within one context.

(Nhiều hệ thống Trí tuệ Nhân tạo có thể cùng cộng tác trong một ngữ cảnh.)

5. Every context contributes to the evolution of AHI.

(Mọi ngữ cảnh đều góp phần vào sự tiến hóa của AHI.)
