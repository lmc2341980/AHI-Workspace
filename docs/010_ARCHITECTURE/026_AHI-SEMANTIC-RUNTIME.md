# Purpose (Mục đích)

This document defines the Semantic Runtime (Môi trường Thực thi Ngữ nghĩa) of the Artificial Hybrid Intelligence Workspace (AHI-Workspace - Không gian làm việc Trí tuệ Nhân tạo Lai).

(Tài liệu này định nghĩa Môi trường Thực thi Ngữ nghĩa của Không gian làm việc Trí tuệ Nhân tạo Lai.)

The Semantic Runtime is the execution environment responsible for understanding meaning instead of processing plain text.

(Môi trường Thực thi Ngữ nghĩa là môi trường chịu trách nhiệm hiểu ngữ nghĩa thay vì chỉ xử lý văn bản.)

It transforms every interaction into Semantic Objects (Đối tượng Ngữ nghĩa), Semantic Relationships (Quan hệ Ngữ nghĩa) and Semantic Actions (Hành động Ngữ nghĩa).

(Nó chuyển mọi tương tác thành Đối tượng Ngữ nghĩa, Quan hệ Ngữ nghĩa và Hành động Ngữ nghĩa.)

# Responsibilities (Trách nhiệm)

The Semantic Runtime shall:

(Môi trường Thực thi Ngữ nghĩa phải:)

- Parse meaning instead of sentences.

(Phân tích ngữ nghĩa thay vì câu chữ.)

- Normalize concepts from different languages.

(Chuẩn hóa khái niệm từ nhiều ngôn ngữ.)

- Generate Semantic Objects.

(Tạo Đối tượng Ngữ nghĩa.)

- Build Semantic Relationships.

(Xây dựng Quan hệ Ngữ nghĩa.)

- Synchronize with the Knowledge Graph (Đồ thị Tri thức).

(Đồng bộ với Đồ thị Tri thức.)

- Support unlimited languages.

(Hỗ trợ không giới hạn ngôn ngữ.)

- Preserve semantic consistency.

(Bảo đảm tính nhất quán ngữ nghĩa.)

# Core Components (Các Thành phần Lõi)

## Semantic Parser (Bộ Phân tích Ngữ nghĩa)

Converts natural language into semantic structures.

(Chuyển đổi ngôn ngữ tự nhiên thành cấu trúc ngữ nghĩa.)

---

## Semantic Normalizer (Bộ Chuẩn hóa Ngữ nghĩa)

Maps equivalent concepts across multiple languages.

(Ánh xạ các khái niệm tương đương giữa nhiều ngôn ngữ.)

---

## Semantic Identifier Manager (Bộ Quản lý Định danh Ngữ nghĩa)

Assigns permanent Semantic Identifiers (Định danh Ngữ nghĩa) to Semantic Objects.

(Gán Định danh Ngữ nghĩa vĩnh viễn cho các Đối tượng Ngữ nghĩa.)

---

## Semantic Validator (Bộ Kiểm chứng Ngữ nghĩa)

Verifies semantic consistency before knowledge is stored.

(Kiểm tra tính nhất quán ngữ nghĩa trước khi lưu trữ tri thức.)

---

## Semantic Action Generator (Bộ Sinh Hành động Ngữ nghĩa)

Transforms semantic understanding into executable actions.

(Chuyển đổi hiểu biết ngữ nghĩa thành các hành động có thể thực thi.)

# Design Principles (Nguyên tắc Thiết kế)

## Principle 001 (Nguyên tắc 001)

Meaning is more important than language.

(Ngữ nghĩa quan trọng hơn ngôn ngữ.)

---

## Principle 002 (Nguyên tắc 002)

Every Semantic Object has one permanent Semantic Identifier (Định danh Ngữ nghĩa).

(Mỗi Đối tượng Ngữ nghĩa có một Định danh Ngữ nghĩa vĩnh viễn.)

---

## Principle 003 (Nguyên tắc 003)

Languages are interchangeable representations.

(Ngôn ngữ chỉ là các cách biểu diễn khác nhau.)

---

## Principle 004 (Nguyên tắc 004)

The Semantic Runtime continuously enriches the Knowledge Graph (Đồ thị Tri thức).

(Môi trường Thực thi Ngữ nghĩa liên tục làm giàu Đồ thị Tri thức.)

---

## Principle 005 (Nguyên tắc 005)

The Semantic Runtime shall evolve without breaking existing knowledge.

(Môi trường Thực thi Ngữ nghĩa phải tiến hóa mà không làm phá vỡ tri thức hiện có.)
