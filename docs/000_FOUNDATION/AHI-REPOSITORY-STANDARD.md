# Purpose (Mục đích)

This document defines the mandatory standard for all repositories within the Artificial Hybrid Intelligence (AHI - Trí tuệ Nhân tạo Lai) ecosystem.

(Tài liệu này quy định tiêu chuẩn bắt buộc cho tất cả Repository trong hệ sinh thái Trí tuệ Nhân tạo Lai.)

---

# Rule 001 (Quy tắc 001)

Every repository name shall start with **AHI-** unless there is a justified exception.

(Mọi Repository phải bắt đầu bằng tiền tố **AHI-**, trừ khi có lý do đặc biệt.)

Examples (Ví dụ):

- AHI-Workspace
- AHI-Core
- AHI-Language
- AHI-Applications

---

# Rule 002 (Quy tắc 002)

Every repository shall use English as the primary naming language.

(Mọi Repository sử dụng tiếng Anh làm ngôn ngữ đặt tên chính.)

Descriptions and documents shall be bilingual.

(Mô tả và tài liệu phải song ngữ Anh - Việt.)

---

# Rule 003 (Quy tắc 003)

Every repository shall contain at least the following root files.

(Mọi Repository phải có tối thiểu các tệp gốc sau.)

```text
README.md
LICENSE
CHANGELOG.md
ROADMAP.md
CONTRIBUTING.md
CODE_OF_CONDUCT.md
SECURITY.md
```

---

# Rule 004 (Quy tắc 004)

Every repository shall contain the following root folders.

(Mọi Repository phải có tối thiểu các thư mục sau.)

```text
docs/
src/
tools/
.github/
```

---

# Rule 005 (Quy tắc 005)

All architecture documents shall be stored under:

(Mọi tài liệu kiến trúc được lưu tại:)

```text
docs/
```

---

# Rule 006 (Quy tắc 006)

All source code shall be stored under:

(Toàn bộ mã nguồn được lưu tại:)

```text
src/
```

---

# Rule 007 (Quy tắc 007)

All automation scripts shall be stored under:

(Toàn bộ công cụ tự động hóa được lưu tại:)

```text
tools/
```

---

# Rule 008 (Quy tắc 008)

Every repository shall include GitHub configuration files under:

(Mọi Repository phải chứa cấu hình GitHub tại:)

```text
.github/
```

---

# Rule 009 (Quy tắc 009)

Every repository shall inherit all constitutions and standards previously approved within the AHI ecosystem.

(Mọi Repository phải kế thừa tất cả Hiến pháp và Tiêu chuẩn đã được phê duyệt trong hệ sinh thái AHI.)

---

# Rule 010 (Quy tắc 010)

GitHub shall be the only official Source of Truth (Nguồn sự thật chính thức).

All documents, source code and project evolution shall be synchronized through GitHub.

(Toàn bộ tài liệu, mã nguồn và quá trình tiến hóa của dự án phải được đồng bộ thông qua GitHub.)
