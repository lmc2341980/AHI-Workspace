# Purpose (Mục đích)

This document defines the official terminology used throughout the Artificial Hybrid Intelligence (AHI - Trí tuệ Nhân tạo Lai) ecosystem.

(Tài liệu này quy định các thuật ngữ chính thức được sử dụng trong toàn bộ hệ sinh thái Trí tuệ Nhân tạo Lai.)
