# Purpose (Mục đích)

This document defines the mandatory working protocol between the AHI Founder and ChatGPT.

(Tài liệu này định nghĩa quy trình làm việc bắt buộc giữa Người sáng lập AHI và ChatGPT.)

All future repositories, documents, source code and conversations shall follow this document unless superseded by a newer revision.

(Mọi Repository, tài liệu, mã nguồn và hội thoại sau này đều phải tuân theo tài liệu này nếu chưa có phiên bản mới thay thế.)

---

# Rule 001 (Quy tắc 001)

ChatGPT shall prioritize implementation before discussion.

(ChatGPT phải ưu tiên triển khai trước khi thảo luận.)

---

# Rule 002 (Quy tắc 002)

Every response shall produce GitHub Ready content.

(Mọi câu trả lời phải sinh nội dung sẵn sàng đưa lên GitHub.)

---

# Rule 003 (Quy tắc 003)

Every file shall contain both English and Vietnamese.

(Mọi tài liệu đều phải có tiếng Anh và tiếng Việt.)

---

# Rule 004 (Quy tắc 004)

Every English abbreviation shall be expanded with its English full name and Vietnamese meaning when first used in each document.

(Mọi từ viết tắt tiếng Anh khi xuất hiện lần đầu trong mỗi tài liệu phải ghi đầy đủ tên tiếng Anh và chú thích tiếng Việt.)

Example:

API (Application Programming Interface – Giao diện lập trình ứng dụng)

# Rule 005 (Quy tắc 005)

GitHub is the only Source of Truth.

(GitHub là nguồn sự thật duy nhất.)

---

# Rule 006 (Quy tắc 006)

Documentation First.

(Tài liệu trước.)

Implementation Second.

(Triển khai sau.)

Evolution Always.

(Luôn tiến hóa.)

# Rule 007 (Quy tắc 007)

Every GitHub Ready block shall contain only the content to be pasted into GitHub.

(Mọi vùng GitHub Ready chỉ được chứa nội dung cần dán lên GitHub.)

(Không được đưa phần giải thích vào vùng GitHub Ready.)
