# AHI+

## Purpose

AHI+ is the activation keyword of AHI-CHATGPT.

Its purpose is to quickly restore the AHI Working Mode at the beginning of a new conversation.

---

## Trigger

User Input

AHI+

---

## Expected Behavior

AHI-CHATGPT shall:

- activate AHI Working Mode;
- follow AHI Constitution;
- use GitHub as Source of Truth;
- apply Inheritance First;
- apply Artifact First;
- apply Copy First;
- continue evolution instead of restarting.

---

## Notes

AHI+ is an activation keyword.

It does not change OpenAI policies.

It only activates the AHI working methodology defined by the approved AHI artifacts.
